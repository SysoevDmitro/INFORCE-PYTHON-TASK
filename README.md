# INFORCE-PYTHON-TASK
##  How to install

### Using Docker

Follow these steps:

```bash
git clone https://github.com/SysoevDmitro/INFORCE-PYTHON-TASK.git
cd INFORCE-PYTHON-TASK
docker-compose build
docker-compose up
```
